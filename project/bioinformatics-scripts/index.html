Bioinformatic scripts

6 May, 2019
Link to code repositories

An assortment of smallish scripts and packages that I’ve created during my PhD for various bioinformatics tasks.

Biomolecular id remapper

A simple Python script to remap protein or gene identifiers into different formats via the UniProt API. The script automatically splits big queries into smaller chunks and tries to retrieve the results multiple times upon failure. Documentation is available here .

goscripts

A Python package that provides functionality for handling Gene Ontology (GO) data (e.g. parsing and manipulating .obo and .gaf files), including a ready to use script to perform GO enrichment test.

Installation instructions are available here and the full (Sphinx-generated) documentation for this project is available here .
